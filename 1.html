HTML이란 무엇인가?

Hybertext Markup Language (HTML) is the standard markup language for creating web pages and web applications. Web browsers receive HTML documetns from a web server or from local storage and render them into multimedia web pages. HTML describes the structure of a web page semantically and origianlly included cues for the appearnce of the document.
